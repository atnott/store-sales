# Система магазина «Shop Manager»

Программное решение для автоматизации процесса покупки и учета товаров в магазине, разработанное в рамках самостоятельной работы по БД

## 📋 Техническое задание
Согласно требованиям, приложение реализует:
*   **Спроектированную базу данных**, автоматизирующую процесс покупки и учета
*   **Учет товаров**: описание (название, категория), цена за единицу, количество на складе
*   **Логику продаж**: покупка нескольких товаров разных категорий с проверкой остатка на складе
*   **Расчет стоимости**: отражение даты продажи, id чека, перечня купленного и количества
*   **Аналитику**: отчет по количеству проданных товаров и выручке за выбранную дату
*   **GUI**: Графический интерфейс для взаимодействия с пользователем

## 🛠 Технологии
*   **Python 3.10+**
*   **SQLite3**
*   **Tkinter**
*   **Type Hinting**
